PUBLICAR EM PRODUÇÃO
    - No Publish, utilize o perfil Publish_PRD.
    - Observe que para publicar em PRD está setado como Release


PUBLICAR EM HML
    - No Publish, utilize o perfil Publish_hml.
    - Observe que para publicar em HML, está setado como Debug


Para debugar localmmente,  start a aplicação em Modo Debug utilizando o perfil "Development".
